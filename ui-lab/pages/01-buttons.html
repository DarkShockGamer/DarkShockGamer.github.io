Buttons & Actions

All interactive button components — primary, secondary, danger, ghost, icon, loading, groups, segmented controls, morphing gradients, FABs, and CTAs. Click any button to hear the sound and see the ripple effect.

11 components ✅ Sound-enabled ♿ Accessible
1 Primary Button
btn-primary

The main call-to-action button. Used for the most important action on a page. High visual weight with the brand sky-blue fill.

<button class="btn-primary btn-ripple" data-sound="click">Primary Button</button>
2 Secondary Button
btn-secondary

Lower visual weight alternative. Pairs well with a primary button for cancel / confirm patterns.

<button class="btn-secondary btn-ripple" data-sound="click">Secondary Button</button>
3 Danger Button
btn-danger

Destructive action button. Use for deletes, removals, or irreversible actions that need user attention.

<button class="btn-danger btn-ripple" data-sound="error">Delete Account</button>
4 Ghost Button
btn-ghost

Outlined transparent button. Subtle — great for secondary actions alongside a primary CTA, or in toolbars.

<button class="btn-ghost btn-ripple" data-sound="click">Ghost Button</button>
5 Icon Buttons

6 Loading Button
loading state

Triggers an async loading state on click. The spinner replaces the label and the button is disabled until the operation completes (2.5 s demo). Prevents double-submit.

Click either button to see the loading state — it auto-resets after 2.5 s.

// JavaScript: toggle loading state btn.addEventListener('click', () => { btn.disabled = true; btn.innerHTML = '<span class="spinner"></span> Loading…'; setTimeout(() => { btn.disabled = false; btn.innerHTML = originalLabel; }, 2500); });

8 Segmented Control
seg-control

9 Morphing Gradient Button
btn-morph

Animated multi-stop gradient button. The gradient continuously shifts position, creating a living, pulsing effect. Hover accelerates the animation.

<button class="btn-morph btn-ripple" data-sound="morph">✨ Activate</button>
